Cada projeto listado aqui representa uma etapa da minha jornada: os primeiros experimentos com HTML e CSS, a transição para JavaScript, a adoção do React e, mais recentemente, a integração com APIs de inteligência artificial e TypeScript.

## Minha jornada como desenvolvedor

### Fase 1 — Fundamentos (HTML, CSS, JS puro)
Comecei do zero: estruturas HTML semânticas, CSS com animações e layouts responsivos, e JavaScript para manipulação de DOM. O projeto **Peanut Drinks** é o maior exemplo dessa fase — um e-commerce multi-página completo, com carrinho de compras, checkout e persistência via `localStorage`, tudo sem nenhum framework.

### Fase 2 — Componentização com React
A descoberta do React foi um divisor de águas. Passei a pensar em interfaces como componentes reutilizáveis, com estado gerenciado via Hooks. Projetos como o **Blog React** e **Detalhes Uniformes** mostram essa transição: código mais organizado, separação de responsabilidades e uma base escalável.

### Fase 3 — Projetos complexos e APIs externas
Com a base consolidada, comecei a integrar APIs externas e a construir aplicações com lógica mais robusta. O **GeoFinder** é o ponto alto dessa fase: usa a API do **Google Gemini (IA)** para identificar localizações geográficas a partir de imagens e exibe os resultados em um mapa interativo via OpenStreetMap. Também está disponível como extensão do Chrome.

### Fase 4 — Full-Stack e TypeScript
O **Dashboard Administrativo** marca minha entrada no desenvolvimento full-stack. Construído com React, TypeScript, tRPC e SQLite, exibe métricas em tempo real, gráficos dinâmicos e tabelas filtráveis — com uma arquitetura que vai além do front-end.
