# Portfólio 2026 — Lucas Brandão

Meu portfólio pessoal feito em React, reunindo os projetos que desenvolvi ao longo da minha jornada.

🌐 **[lucasbrandao-portfolio.vercel.app](https://lucasbrandao-portfolio.vercel.app/)**

---

## Stack do portfólio

- **React 18** com Hooks
- **Framer Motion** — scroll reveals, parallax, marquee e micro-interações
- **Vite** para build
- **CSS3** com variáveis e animações
- Tema dark/light com persistência
- Formulário de contato via **Formspree**
- Deploy na **Vercel**
